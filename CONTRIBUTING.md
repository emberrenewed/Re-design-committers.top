# Contributing to Re-design committers.top

Thanks for helping improve the project.

## Local checks

- Run `go test ./...`.
- Run `go build ./...`.

## Contribution guidelines

- Keep pull requests small and focused.
- Preserve the existing project style unless a change intentionally redesigns it.
- Explain ranking, API, or deployment changes clearly.
- Never commit tokens, local environment files, or generated output.
